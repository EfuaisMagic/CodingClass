Class (.) Group Styles

In the HTML document find many elements you want to have the same style.

< p > I Am One Of A Kind < /p >
< p > Why Can't We Be Friends < /p >
< p >...And Love Dares You To Care For The People On the Edge Of the Night < /p >

Now place class = "name" in the containing tags. The name has to be the same for all.

< p = "name" > I Am One Of A Kind < /p >
< p = "name" > Why Can't We Be Friends < /p >
< p = "name" >...And Love Dares You To Care For The People On the Edge Of the Night < /p >

Now go to the CSS and type the name you chose preceded by a perid . Create styles for .name

.name {
color: pink;
font-family: fantasy;
background-color: orange;
text-decoration: wavy;
text-align: center;
}

The styles for the name are applied to all the element  and should now show in browser.

< p = "name" > I Am One Of A Kind < /p >
< p = "name" > Why Can't We Be Friends < /p >
< p = "name" >...And Love Dares You To Care For The People On the Edge Of the Night < /p >
