ID (#) uniqe styles

In the HTML document single out one element.

< p > I Am One Of A Kind < /p >

Now place id = "name" in the containing tag. The name should be one you will not use again.

< p id = "name" > I Am One Of A Kind < /p >

Now go to the CSS and type the name you chose preceded by # . Create styles for #name

#name {
color: pink;
font-family: fantasy;
background-color: orange;
text-decoration: wavy;
text-align: center;
}

The styles for the name are applied to the element and should now show in browser.
