text-align

aligning text is not relative to the document but relative to whatever container holds the text

  • text-align:left;
  • text-align:right;
  • text-align:center;
  • text-align:justify;
  • text-align:justify-all;
  • text-align:start;
  • text-align:end;
  • text-align:match-parent;

font-weight

Not all fonts have as many weights as listed, especialy custom fonts

  • fontweight:normal;
  • fontweight:bold;
  • fontweight:light;
  • fontweight:bolder;
  • fontweight:100-900;

text-decoration

Not all fonts have as many weights as listed, especialy custom fonts

  • text-decoration:wavy;

  • text-decoration:double;

line-height

Does not control the height of text but the height of the line text sits on

  • line-height:normal
  • line-height:2.5;;
  • line-height:3em';
  • line-height:150%;;
  • line-height:32px;;
